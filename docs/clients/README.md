# Client setup guides

This folder contains configuration and setup guides for specific clients using the ChattyBot platform. The platform is **generalized and modular**: the same quoting, photo assessment, industries & protocols, and multi-channel delivery work for any contractor (roofing, plumbing, HVAC, etc.).

Each client guide describes how to configure the **same** platform features for that business (industry, job types, pricing, personality, booking link, etc.).

- [Ryan's Roofing / Raffy AI](./ryans-roofing-raffy-setup.md) – Example setup for a roofing contractor in Ontario (Durham Region).
